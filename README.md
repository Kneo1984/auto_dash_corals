# AUTO_DASH_CORALS  Deep Sea Dashboard Automator

Ein vollständiges, lokal ausführbares PowerShell-Skript zur Automatisierung von:

- MySQL  CSV-Export
- Power BI Refresh & PDF-Export
- Ghostscript PNG-Erstellung
- PowerPoint-Slide-Generierung
- Projektintegration für Uni/Abgabe

## Voraussetzungen
- MySQL Server 8
- Ghostscript
- Power BI Desktop
- PowerShell 5+
- \pbixrefresher.exe\ (optional)
